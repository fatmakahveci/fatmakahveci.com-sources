- `tailwindcss` applies one `CSS` rule at a time.
- It uses predefined classes.
- Install [`node.js`](https://nodejs.org/en)
- Run `npx tailwindcss init`
- `npx tailwindcss -i <dir>/<css_file_name>.css -o <dir>/<css_file_name>.css --watch`
