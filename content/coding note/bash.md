## basename

- It strips paths from filenames:
    - `$ >> basename <dir>/<file_name.ext>`
        % <file_name.ext>

- It can also strip a suffix (e.g., extension):
    - `$ >> basename -s <.ext> <dir>/<filename.ext>`
        % <file_name>
